🧪 Journey 1: New User Registration & Dashboard

Manual Test Validation for Alpaca Streaming Integration

⏳ Test Status: PENDING - Ready to Start

🎯Test Objective

Verify that new users can successfully register and immediately see real-time stock prices on the dashboard with proper data source indicators. This tests the complete onboarding flow and confirms that Alpaca streaming integration works for first-time users.

Note: This journey tests the most critical path for new user acquisition. Any failure here represents a critical business impact.

🔧Environment Setup

Health Check: GET http://localhost:8080/api/health Expected: {"status": "Healthy", "alpacaStreaming": true}

📋Test Steps

Step 1: Navigate to Registration Page

Action: Open web browser and navigate to http://localhost:3000

Expected: Landing page loads, "Register" link visible

Step 2: Click Register Link

Expected: Registration form appears with fields: Email, Password, Confirm Password, First Name, Last Name

Action: Enter test credentials:

Step 4: Submit Registration

Action: Click "Register" or "Sign Up" button

Expected: Request sent to POST /api/v1/auth/register, success response received

Expected API Response:

  • HTTP 200/201 status
  • Response contains: accessToken, refreshToken, user object
  • User object has: id, email, firstName, lastName, isActive: true

Step 5: Verify Auto-Login

Action: Observe page behavior after registration

Expected: User automatically logged in, redirected to Dashboard

Action: Wait for dashboard to fully load

Expected: Dashboard displays with accordion sections: Crypto, Stocks, etc.

Step 7: Check Stock Prices Display

Action: Expand the "Stocks" accordion section

Expected: Stock symbols display with current prices (AAPL, MSFT, GOOGL, etc.)

Verify Each Stock Item Shows:

  • Symbol name (e.g., AAPL)
  • Current price (numeric value)
  • Price change (+ or -)
  • Percentage change
  • Data source badge ("Live" or "Delayed")

Step 8: Verify Data Source Badges

Action: Check for data source indicators on stock items

Expected: If Alpaca streaming active, badges show "Live" (green). If Yahoo fallback, show "Delayed" (yellow/orange)

Action: Keep dashboard open for 60 seconds, watch for price changes
